Yo! Kenalin, **Gabutz Game** — sesuai namanya, game ini diciptakan khusus buat nemenin kamu pas lagi bener-bener mati gaya alias gabut total. 

Game ini mengusung tema retro 8-bit tapi dibalut dengan UI flat kotak monokrom yang super modern dan estetik. Nggak cuma itu, suasananya makin dapet dengan iringan musik synthwave dan chiptune yang udah di-tweak biar makin nendang di telinga.

### 🎮 Ada Apa Aja di Sini?
- **UI Flat & Minimalis:** Gak ada lagi shadow-shadow lebay, semuanya serba kotak, flat, hitam-putih, simpel, tapi tetep keliatan mahal.
- **Toko Gabutz:** Bisa beli kustomisasi lucu-lucuan kayak efek ekor neon, ledakan epik, sampai ganti irama BGM.
- **Beragam Mode Main:** Ada mode Fast, Survival, dan Sandbox (buat yang cuma mau sekadar pencet-pencet doang tanpa mikir).
- **Dapur Pacu:** Dibangun pakai Flutter + Riverpod, dikawinin sama Phaser JS buat nanganin permainannya biar performanya lancar jaya.

### 🚀 Cara Cobain
Kalo lo mau coba utak-atik atau nge-build sendiri:
1. Pastiin udah install Flutter di laptop/PC lo.
2. Clone repo ini.
3. Buka terminal terus ketik `flutter pub get` buat nyedot semua package-nya.
4. Habis itu gas jalanin pake `flutter run`.
